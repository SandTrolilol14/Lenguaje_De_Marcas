# JavaScript Projects

This repository contains two web development projects focusing on DOM manipulation, CSS animations, and JavaScript logic.

## Project 1: Automatic Chessboard
A dynamically generated 8x8 chessboard.
* Built using nested loops in JavaScript.
* Uses mathematical conditions to alternate square colors.
* Added starting pieces to specific rows.

## Project 2: Animation Mini-game
A simple side-scrolling jump game.
* Uses CSS keyframes to animate a moving obstacle and a jumping character.
* Uses JavaScript event listeners to trigger the jump animation when the Spacebar is pressed.
* Includes collision detection using setInterval and coordinate calculations.
* Game Over alert when the character collides with the obstacle.

## Technologies Used
* HTML
* CSS
* JavaScript
